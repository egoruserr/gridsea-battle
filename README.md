# GridSea Battle

Обучающее веб-приложение для изучения CSS Grid в формате морского боя.

## Описание

Пользователь изучает свойства CSS Grid, размещая корабли на игровом поле 6x6. Каждый уровень содержит теоретическую справку и практическое задание. Пользователь пишет CSS-код в редакторе, нажимает "Проверить" и получает обратную связь.

## Технологии

- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript (ES6)
- Модульная архитектура

## Уровни

1. display: grid
2. grid-template-columns и grid-template-rows
3. Размещение корабля в клетке C4
4. Вертикальный корабль в A1-A2
5. Горизонтальный корабль в D3-E3
6. Вертикальный корабль в D3-D4
7. minmax() и адаптивная ширина колонок
8. Три корабля разной формы
